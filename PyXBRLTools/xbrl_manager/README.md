# BaseXbrlManager

XBRLディレクトリの解析をするための基底クラスです。

このクラスを継承したクラスには下記の機能を提供します。

## 機能

* ファイルの一覧
* 書類の種類を表示
* ファイルを抽出

## メソッド

| メソッド           | 説明                                       |
| ------------------ | ------------------------------------------ |
| to_filelist        | ディレクトリ内のファイルをlistで返します。 |
| xbrl_type          | XBRLの種類を確認します。                   |
| to_schema_filelist | 関係ファイルのリストを返します。           |
| to_htmlbase_files  | htmlbaseファイルのリストを返します。       |
